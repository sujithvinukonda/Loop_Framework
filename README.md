# Selenium Cucumber BDD Automation Framework

This project is a robust and scalable test automation framework built with:

- **Selenium WebDriver**
- **Cucumber BDD**
- **Java**
- **JUnit**
- **Extent Reports**
- **Page Object Model (POM)**
- **Apache POI (Excel-driven test data)**
- **Log4j Logging**
- **WebDriverManager**
